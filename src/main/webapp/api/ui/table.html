表格 : @p.table

基本实例：

封装HTML<table>标签，并对前端样式进行封装，实现数据异步获取，并填充表格。 对于<@p.column>标签的使用请查阅@p.column标签使用说明。

ID 用户名 登录名 密码 住址 手机 
					
						
<@p.table id="userTable" url="${base}/user/list.json">
	<@p.column field="uid" code="ID" />
	<@p.column field="userName" code="user.userName" />
	<@p.column field="loginName" code="user.loginName" />
	<@p.column field="password" code="user.password" />
	<@p.column field="address" code="user.address" />
	<@p.column field="phone" code="user.phone" />
</@p.table>

参数说明：

在模板页的任意位置调用@p.table标签并传入相应参数即可展现Bootstarp风格的<table>表格，并异步远程读取数据填充表格。

注意

在以下参数中已经封装默认值的，在使用过程中可以根据需求来改变参数的传值以实现不同的表格效果。参数名应注意区分大小写。

表格工具栏实例

在<@p.table>中添加 toolbar="#elementId" 参数，并传入你所要添加的元素id，即可为数据表格增加“工具栏”。

ID 用户名 登录名 密码 住址 手机 
					
	<@m.buttonArea id="button-bar">
		<@p.button code="resource.add" id="add" name="add" />
		<@p.button code="resource.edit" id="edit" name="edit" />
		<@p.button code="resource.delete" id="delete" name="delete" />
	</@m.buttonArea>

	<@p.table id="userTable" url="${base}/user/list.json" toolbar="#button-bar">
		<@p.column field="uid" code="ID" />
		<@p.column field="userName" code="user.userName" />
		<@p.column field="loginName" code="user.loginName" />
		<@p.column field="password" code="user.password" />
		<@p.column field="address" code="user.address" />
		<@p.column field="phone" code="user.phone" />
	</@p.table>
