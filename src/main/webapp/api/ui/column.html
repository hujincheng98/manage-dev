表格 : @p.column

参数说明：

在@p.table标签中嵌套使用。

注意

在以下参数中已经封装默认值的，在使用过程中可以根据需求来改变参数的传值以实现不同的表格效果。参数名应注意区分大小写。

变量 类型 默认值 参考值 是否选填 说明
radio Boolean false true / false 是 设置为True显示单选框，单选框列有固定的宽度。
checkbox Boolean false true / false 是 设置为True显示复选框，复选框列有固定的宽度。
field String - - 否 列标识名称。
title String - - 是 表头标题。
class String - - 是 列的样式名称。
align String - left / right / center 是 定义列的水平对齐方式，只能为："left", "right" 和 "center"。
halign String - left / right / center 是 定义表头的水平对齐方式，只能为："left", "right" 和 "center"。
valign String - top / middle / bottom 是 定义列的垂直对齐方式，只能为："top", "middle" 和 "bottom"。
width Number - - 是 列的宽度，假如没有定义，将根据内容自适应宽度。
sortable Boolean true true / false 是 设置为True允许对该列进行排序。
order String asc asc / desc 是 定义列排序的顺序，只能为“asc”和“desc”。
visible Boolean true true / false 是 设置为False隐藏该列。
switchable Boolean true true / false 是 设置为False禁用切换列的显示/隐藏。
clickToSelect Boolean true true / false 是 设置为True时点击行即可选中单选/复选框。
formatter Function limit limit 是 列的格式化方法，包含3个参数：
value: 该单元格的数据。
row: 该行的数据。
index: 该行的index。
events Object - - 是 当你使用格式化方式时，可以对该列的元素进行事件监听。包含三个参数：
event: jQuery 事件。
value: 该单元格的数据。
row: 该行的数据。
index: 该行的index。
sorter Function - - 是 可对列进行自定义排序，包含两个参数：
a: 第一个值。
b: 第二个值。
cellStyle Function - - 是 列样式格式化方法，有三个参数：
row: 行记录的数据。
value: 该单元格的数据。
index: 行数据的 index。
支持 classes 或者 css.
