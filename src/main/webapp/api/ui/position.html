表格 : @m.position

表格工具栏实例

在<@p.table>中添加 toolbar="#elementId" 参数，并传入你所要添加的元素id，即可为数据表格增加“工具栏”。

  1. 首页
  2. 用户管理
  3. 用户列表
					
<@m.position list=["global.index", "user.management", "user.list"] />

参数说明：

在页面顶部位置插入@m.position标签即可显示当前位置。

注意

在以下参数中已经封装默认值的，在使用过程中可以根据需求来改变参数的传值以实现不同的表格效果。参数名应注意区分大小写。

变量 类型 默认值 参考值 是否选填 说明
list Array - ["golbal.index", "user.manager","user.list"] 否 设置当前页面位置信息。参数传值为国际化资源
